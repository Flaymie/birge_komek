# Бірге Көмек — P2P EdTech платформа для школьников

## О проекте

**«Бірге Көмек»** (каз. "Поможем вместе") — это EdTech веб-платформа для peer-to-peer обучения, где школьники могут помогать друг другу с учебой в режиме реального времени.

**Ключевая идея:** Создать безопасную среду, где ученики получают не готовые ответы, а помощь в понимании материала от сверстников.

## Технологический стек

| Слой | Технологии |
|------|------------|

## Ключевые возможности

### Для пользователей
-   **Заявки на помощь** — создание запросов с файлами и описанием проблемы
-   **Система откликов** — хелперы предлагают помощь, ученик выбирает
-   **Real-time чат** — приватное обсуждение между учеником и хелпером
-   **Рейтинг и отзывы** — система репутации для хелперов
-   **Telegram-интеграция** — вход, уведомления, 2FA

### Для платформы
-   **AI-модерация (Gemini)** — автоматический анализ заявок на попытки списывания
-   **Анти-чит система** — обнаружение мультиаккаунтов и подозрительной активности
-   **RBAC** — система ролей (Ученик → Хелпер → Модератор → Админ)
-   **Rate limiting** — защита от DDoS и брутфорса

## Безопасность

-   **JWT-аутентификация** с коротким временем жизни токенов
-   **Валидация данных** — защита от XSS и NoSQL-инъекций
-   **IP-верификация** — подтверждение через Telegram при входе с нового устройства
-   **Секреты в `.env`** — ничего не попадает в репозиторий

## Кейсы использования

**Срочная помощь:** Ученица создаёт заявку за час до контрольной → через 3 минуты хелпер объясняет тему в чате → успешная контрольная.

**Заработок рейтинга:** Сильный ученик помогает другим → получает отзывы и высокий рейтинг → становится топ-хелпером.
